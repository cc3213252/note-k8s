## 虚拟机和容器区别

虚拟机：资源占用多、冗余步骤多、启动慢。 
容器：必要的内核，centos基础镜像就170M

资源角度：  
虚拟机：一整套  
容器：Linux容器，缩写LXC，小巧的linux必须的内核支撑我的软件  

虚拟化：  
虚拟机：使用硬件实现的虚拟化  
容器：复用本地主机操作系统，系统层面实现虚拟化
